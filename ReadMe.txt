This is HerdProtectChecker version 1.1
============================
Script checking correctness of file characteristics on the "herdprotect.com"
============================
To successfully run script you need to:
- install Python 2.+ (correct work with python 3.+ is not verified)

- install Beautiful Soup by by using command line:
		> pip install beautifulsoup4
	website: http://www.crummy.com/software/BeautifulSoup/

- run script using command line with argument (use path to folder containing files need to be checked as argument)
	example:
		>main.py testfolder
	or
		>main.py C:\testfolder
